## Project Overview
This repository contains a technical and philosophical audit of the Git version control system. It explores Git's origin (the BitKeeper crisis), its GPL v2 licensing, and its impact on the FOSS ecosystem.

## Shell Scripts Included
Inside the `scripts/` folder, there are 5 functional shell scripts that demonstrate practical Linux skills and automation related to open-source workflows:
1. `script1_identity.sh`: Generates a System Identity Report.
2. `script2_package.sh`: Inspects the FOSS package (Git) and provides a philosophy note.
3. `script3_disk.sh`: Audits disk space and directory permissions.
4. `script4_log.sh`: Analyzes log files for specific keywords.
5. `script5_manifesto.sh`: Interactively generates a personalized Open Source Manifesto.

## How to Run
To execute these scripts on a Linux system, use the following commands:
1. Navigate to the scripts directory: `cd scripts`
2. Make all scripts executable: `chmod +x *.sh`
3. Run any script, for example: `./script1_identity.sh`
